## Remaining Architecture Cut

The hard unfinished part is not history or DKT model shape. It is source-of-truth ownership: MiniCut still has a command/patch worker registry and an observable React store, while weather runs the DKT model runtime as the worker authority and syncs that runtime to the page. The next migration should be one focused PR-sized slice that introduces the worker DKT model runtime and sync stream transport before deleting the old patch authority.
